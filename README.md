# Enterprise Document Intelligence Agent (Agentic AI)

## 🚀 Overview
This project is an Agentic AI system that processes documents and dynamically decides actions such as question answering, summarization, and MCQ generation.

## 🧠 Features
- Multi-PDF upload
- RAG-based Question Answering
- Document Summarization
- MCQ Generation
- Agent-based decision system
- Chat history

## 🏗️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq LLM

## ⚙️ How it Works
1. Documents are converted into embeddings
2. Stored in FAISS vector DB
3. Agent decides task based on user query
4. LLM generates response

## 🎯 Use Cases
- Education
- Research
- Enterprise document search
